Canonical Identity documentation
================================

Canonical Identity represents a suite of identity and access management (IAM) solutions provided by Canonical,
encompassing the Canonical Identity Platform alongside a growing stack of additional extensions and integrations.
It is designed to be a highly modular, secure, and scalable ecosystem that provides centralized authentication,
authorization, and access governance controls across diverse infrastructures.
