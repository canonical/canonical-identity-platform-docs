# Charmed OAuth2 Proxy
OAuth2 Proxy is a Kubernetes charmed operator for [OAuth2 Proxy](https://oauth2-proxy.github.io/oauth2-proxy/) - a Cloud Native Computing Foundation (CNCF) Sandbox project.
It is a reverse proxy and a static file server that authenticates users through providers like Google and GitHub, allowing validation by email, domain, or group.
It integrates with the [Canonical Identity Platform](https://charmhub.io/topics/canonical-identity-platform).

The `oauth2-proxy-k8s` charmed operator handles deployment, scaling, configuration, and other operations on top of [Juju](https://juju.is/).
